## 📱 Telefona Ekle (iPhone & Android)
1. Linki telefon tarayıcısında aç
2. **iPhone:** Paylaş butonu → "Ana Ekrana Ekle"
3. **Android:** Menü → "Ana Ekrana Ekle"
4. Uygulama gibi ikonla açılır!
